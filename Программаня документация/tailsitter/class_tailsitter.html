Detailed Description

Class for controlling the tailsitter VTOL type.

The Tailsitter class is responsible for controlling the transition between multicopter (MC) mode and fixed-wing (FW) mode in a tailsitter vehicle, as well as handling the attitude and thrust control during transitions.
